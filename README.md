# phpcodec
PHP Command line encoding &amp; decoding tool

## Requirements

PHP >= 5.4

zlib

msgpack PHP Extension

## Install
1. make
2. make install

   or

   make **PREFIX=/usr/local** install

## Examples

<code>phpcodec -h</code>
